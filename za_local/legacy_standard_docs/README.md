## Legacy Standard Docs

This directory preserves retired standard-document files that were previously shipped from legacy root paths.

Current policy:

- active South African print formats are canonicalized under module-scoped standard-doc paths such as `za_local/sa_payroll/print_format/...`
- files stored here are archival compatibility references only
- this directory must not be treated as an active import source for standard docs

If a legacy document needs to be reintroduced, migrate it back deliberately and ensure there is only one active JSON source for that standard document.
